# 知识博主方法论 — 提示词库

> 用于系统性研究知识博主（博主）的内容，提炼道法术 + 视频结构 + 钩子金句资产 + 多维分析。
> 所有提示词均按 R-T-C 框架（角色-任务-约束）组织，可直接复制使用。

## 前置准备：单条视频蒸馏模板

> 模板完整版已保存至：
> - `<Skill目录>/知识博主-单条视频蒸馏模板.md`
> - `<Skill目录>/知识博主-单条视频蒸馏模板.md`
>
> 每条视频按此模板逐条分析，涵盖：道法术提炼 + 视频结构拆解 + 钩子金句资产化 + **为什么有效** + 补充标签。

### 模板栏目一览

| 栏目 | 内容 | 来源 |
|------|------|------|
| 一、原文 | 去时间戳的完整口播文字 | 自动提取 |
| 二、道法术提炼 | 道（认知）×1-2 + 法（框架）×1-2 + 术（技巧）×1-3，均含原文依据 | 第一步提示词 |
| 三、视频结构拆解 | 6 维度：钩子、主体、金句、结尾、语言、情绪 + 与"道"的关联；钩子和金句必须补齐资产化字段 | 第二步提示词 |
| 四、为什么有效 | 身份叙事、口吻距离感、情绪钩子、记忆点——分析为什么打动人 | **新增维度** |
| 五、补充标签 | 内容矩阵定位 + 选题策略 + 迭代时期 | 补充维度一/三/四 |
| 六、与其他视频的关联 | 同一主题、同一方法、时间呼应的交叉引用 | 为汇总铺路 |
| 七、我的批注 | 个人启发、与自己的关联、待行动 | 留空自填 |

### 直接使用的提示词

完整提示词见模板文件。使用方式：复制提示词 → 在末尾粘贴一条 SRT 字幕纯文字 → AI 按模板逐栏输出 → 保存为 `.md` 文件。

### 批量简化规则

同一主题下内容高度重复的视频，后出现的只填：原文、道法术（仅差异点）、为什么有效（精简版）、关联。其余标注「同【前一条】」。但钩子和金句资产化字段不能写「同前」，每条视频都要独立判断质量、风险和等级。

## 第一步：提炼道法术

```
【角色】你是一位知识萃取专家，擅长从海量内容中提炼认知框架。你对"道法术"三层模型有深刻理解：
- 道 = 底层认知和价值观，一个人相信什么、追求什么、思维方式的根基
- 法 = 方法论和策略框架，可复用的做事原则和决策模型
- 术 = 具体技巧和操作细节，可直接执行的动作指南

【任务】根据我提供的知识博主内容素材，按"道-法-术"三层进行系统性提炼。

【约束】
- 每个层次单独成章，用表格呈现，表格列包含：编号、提炼要点、原文依据（引用原文或概述出处）、我的理解（留空待我填写）
- 道、法、术三者要有明确区分度——如果某条既像道又像法，说明提炼不够精准，需要重新归类
- 每层控制在 5-10 条，宁缺毋滥，只保留反复出现或他明确强调的内容
- 提炼完成后，额外输出一个「核心关键词云」小节，列出他最高频使用的 10-20 个关键词
- 最后附一个「一句话总结」：用一句话概括知识博主的思想内核是什么

【道法术区分示例（校准用）】
- 道：比如「价值投资是认知的变现」——这是底层信念，不是操作方法
- 法：比如「用复利思维做内容，每一篇都是资产的积累」——这是策略框架
- 术：比如「每条视频前 3 秒抛一个反常识观点」——这是具体可执行动作
如果你的归类既像道又像法，说明提炼不够精准，需要重新归类。

【素材不足怎么办】
如果素材少于 5 条，不要强行提炼，先告诉我「素材不足，建议先补充哪些类型的内容」。

【第一次跑完不满意怎么办（迭代追问）】
把你觉得不满意的地方直接告诉我，比如：
- "道这一层太虚了，给我更具体的"
- "有些条目像是凑数的，删掉，只保留核心的"
- "XX 这个观点你没提到，重新补充"
然后我会重新提炼。这一步可以反复迭代到你满意为止。

【要处理的素材如下】
（粘贴你收集的笔记、书摘、视频文字等内容）
```

## 第二步：拆解视频结构

```
【角色】你是一位短视频/中视频内容结构分析师，精通口播类内容的制作方法论。你能够从大量视频中提取出可复用的结构模板。

【任务】分析知识博主的视频创作结构，拆解出一套可复用的结构模板。

【约束】
请从以下 6 个维度逐一拆解，每个维度都要给出具体例子：

1. 开头钩子（前 3-5 秒）：他常用哪几种钩子类型？除了旧分类（提问型/反常识型/故事悬念型/共鸣型……），还要标注心理触发器、开头动作、开头风格、可迁移模板、风险和等级
2. 主体展开结构：他用什么逻辑组织内容？常见模式有哪些？（故事→观点→论证/问题→分析→解法/现象→本质→行动……）
3. 金句投放节奏：金句出现的频率和位置有没有规律？大概几分钟一句？放在段落的什么位置？每条金句要标注功能、句式模式、可用位置、可迁移模板、风险和等级
4. 结尾设计：他用什么方式收尾？（金句收尾/行动号召/留白思考/下期预告……）
5. 语言风格特征：用词偏好、句式长短、口语化程度、是否有标志性口头禅
6. 情绪节奏：整个视频的情绪高低点如何分布？是先抑后扬还是持续输出？

输出格式要求：
- 每个维度输出完后，给一个「可复用模板」，用简洁的框架式写法，方便我写脚本时直接对照
- 最后汇总成一个「知识博主视频结构总模板」，包含完整的一期视频骨架大纲

【样本量要求】
至少基于 3 条不同时期的视频分析，避免以偏概全。如果素材不足 3 条，先说明「样本不足，结论可能有偏差」。

【与道法术的关联】
分析视频结构时，额外回答一个问题：他的视频结构是如何服务于他的「道」的？比如，如果他的道是「长期主义」，那他的视频节奏是不是也在体现「不急不躁」？

【如果你只有文字版】
如果你只看了视频的文字记录、没看原视频，请明确说明，不要猜测画面呈现、剪辑节奏和视觉情绪。

【如果你手头有他的视频链接或文字记录，可以附加在下面】
```

## 新增维度：为什么有效

> 这是对第一、二步的补充。拆了「他说了什么」和「他怎么说的」，还需要拆「他说的方式为什么打动人」。

```
【角色】你是一位传播效果分析师，擅长分析内容为什么能打动人、被记住、被传播。

【任务】分析这条视频为什么有效。

【约束】从以下 4 个维度逐一分析：

1. 身份叙事：他用了什么个人身份/经历来建立信任？是否提到"中专毕业""普通人""我以前也……"等降低距离感的身份标签？
2. 口吻与距离感：说话的感觉像在跟谁聊天？什么词/句式制造了亲近感或稀缺感？（"老伙计""内部会议""我给你讲"）
3. 情绪钩子：这条视频精准击中了观众的什么痛点/痒点/爽点？观众看完会产生什么感觉？
4. 记忆点：看完后最容易记住的是什么？这个记忆点为什么容易被传播？
```

```
【角色】你是一位自媒体变现研究员。

【任务】梳理知识博主的变现路径演进过程。

【约束】
```

## 综合诊断：全部维度汇总

> 在完成以上 5 个补充维度（或至少完成你感兴趣的 3 个）后，使用此提示词做一次综合诊断。

```
【角色】你是一位自媒体战略顾问。

【任务】基于我对知识博主的全部研究笔记，做一次综合诊断。

【约束】
- 请先读取以下笔记（如果存在）：知识博主-道法术蒸馏、知识博主-视频结构拆解、以及 5 个补充维度的分析笔记
- 综合诊断输出以下内容：
  1. 知识博主的核心竞争力是什么？——用一段话概括，不超过 150 字
  2. 他的方法中，哪些是我（用户当前行业、起步基础、可投入时间）可以直接借鉴的？
  3. 哪些是他有条件但我暂时不具备的？（比如团队、资金、平台资源等）——诚实列出差距
  4. 如果我只做 3 件事来逼近他的水平，应该是哪 3 件？——按优先级排序，每件附一句话理由
  5. 一个风险提示：学习知识博主的过程中，最容易掉进的坑是什么？
```

## 常见问题排查

| 遇到的问题 | 可能原因 | 怎么处理 |
|-----------|---------|---------|
| 提炼出来的道法术太虚，像正确的废话 | 素材不够具体，或 AI 没有看到足够多的原始内容 | 补充至少 5 条具体素材（金句 + 场景），重新跑第一步 |
| 视频结构拆解太泛，感觉套在任何一个博主上都行 | 缺少对比参照，AI 只能给出通用模板 | 在提示词里追加：「同时分析另一位风格不同的博主做对照，突出知识博主的独特之处」 |
| 生成的笔记 YAML 格式不对 | Obsidian 对 frontmatter 格式要求严格 | 用第三步的「验证清单」逐条检查，特别是冒号后必须有空格、tags 是列表格式 |
| 双向链接是硬凑的，跟已有笔记没有真实关联 | AI 没有先读知识库就直接写链接 | 先单独跑第三步提示词的最后一段「建立链接前先读库」，再让 AI 补链接 |
| 跑了几步之后，后面的步骤忘了前面的结论 | 每个新对话窗口上下文独立 | 每步完成后把关键结论写回笔记文件，下一步开始时让 AI 先读前一步的笔记 |

## 使用顺序

```
现在立刻做 →
  前置准备（建模板）
  → 第一步（道法术）
  → 第二步（视频结构）
  → 第三步（整合入库）

后续有空再做 →
  维度三（迭代史）← 对你现阶段启发最大
  → 维度四（选题策略）
  → 维度一（内容矩阵）
  → 维度二（变现路径）
  → 维度五（粉丝互动）
```
