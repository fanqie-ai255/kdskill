# 知识博主-单条视频蒸馏模板

> 每条视频按此模板逐条分析。模板涵盖：道法术提炼 + 视频结构拆解 + 钩子金句资产化 + 为什么有效 + 补充标签。
> 分析完成后存入对应分类文件夹。

---

## 使用说明（先看这里）

1. 将下方「提示词」完整复制到对话框中
2. 在提示词末尾粘贴一条 SRT 字幕的纯文字内容
3. AI 会按模板逐栏输出分析结果
4. 将输出保存为 `.md` 文件，文件名 = `【核心观点简化名，≤20字】.md`
5. 存入对应分类文件夹

## 批量处理时的简化

如果同一主题下多条视频内容高度重复，后出现的视频可以只填写：
- 一、原文
- 二、道法术提炼（只记与前一条不同的要点）
- 四、为什么有效（精简版）
- 六、与其他视频的关联

其余栏目标注「同【前一条文件名】」即可。

## 三、视频结构拆解

### 1. 开头钩子（前 3-5 秒）

- **钩子类型（旧分类）**：【提问型 / 反常识型 / 故事悬念型 / 共鸣型 / 直给方法论型 / 内部信息外泄型】
- **心理触发器**：【认知冲突 / 好奇缺口 / 恐惧损失 / 身份代入 / 数字锚定 / 结果承诺 / 社会证明 / 争议挑衅 / 场景条件 / 行动号召 / 权威借力 / 互动测试】
- **开头动作**：【戳痛点 / 抛结果 / 制造悬念 / 亮身份 / 给承诺 / 立冲突 / 借权威 / 下指令 / 讲故事 / 做测试】
- **开头风格**：【扎心 / 温和 / 故事 / 反常识 / 实操 / 可信复盘】
- **原文**：【前 5-20 秒口播原文；如果真正钩子超过 5 秒，保留完整钩子句，不要硬截断】
- **观看理由**：【观众为什么愿意继续看：被戳中 / 想知道答案 / 相信有结果 / 代入身份 / 其他】
- **可信信号**：【数字 / 经历 / 权威 / 具体场景 / 反常识判断 / 无明显可信信号】
- **可迁移模板**：【把这个开头抽象成用户可复用句式，例如：你以为【A】才是问题，其实真正卡住你的是【B】】
- **适合改写到的主题**：【学习 / 赚钱 / AI / 知识库 / 内容创作 / 行动习惯 / 其他】
- **风险**：【低风险 / 强绑定博主人设 / 承诺过强 / 内部黑话 / 卖课感强 / 原句一般但结构可用】
- **等级**：【S / A / B / C / D；S/A 进入主库，B 进结构库，C/D 进次级或反例库】

## 四、为什么有效

> 分析这条视频为什么能打动人、让人愿意看完甚至分享。

### 1. 身份叙事
