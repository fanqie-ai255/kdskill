# CLAUDE.md — 知识博主内容蒸馏工具包

> 这是一套把知识博主的视频内容系统蒸馏为结构化知识库的完整提示词工具包。
> 本工具包可复用于任何知识博主的视频内容蒸馏。

## 使用前注意

- **路径**：文中引用的绝对路径（如 `E:\02 AI\...`）为作者本机示例，使用前请替换为你的实际目录
- **分类**：文中以 9 大分类为例，实际使用时分类数量和名称根据目标博主的内容自行决定，每批 ≤30 条

## 项目概述

帮助用户系统性地研究一位知识博主的内容，提炼道法术 + 视频结构 + 多维分析，最终形成可检索、可复盘、可拿来就用的个人智库。

核心策略：**边学边讲**，学习过程即创作素材，以输出倒逼输入。

## 蒸馏流程

```
准备素材 → 内容分类 → 设计模板 → 批量蒸馏 → 综合汇总（三阶段） → 写入检索规则
```

1. **准备素材**：视频导出为 SRT 字幕文件，纯文本格式
2. **内容分类**：按主题分文件夹，每文件夹 ≤30 条
3. **设计蒸馏模板**：用「单条视频蒸馏模板」跑通 1~2 条，确认样式后冻结
4. **批量蒸馏**：用「批量蒸馏开工指南」多窗口并行，逐批执行
5. **综合汇总-第一阶段**：产出分类深度汇总（每个分类一份，日常检索入口）
6. **综合汇总-第二阶段**：跨分类归纳 → 道法术完整版 + 视频结构完整图谱
7. **综合汇总-第三阶段**：策略落地 → 内容策略总览 + 个人行动指南
8. **写入检索规则**：在知识库 CLAUDE.md 中写入检索规则，日常即可搜索使用

详细操作步骤见 `README-使用说明.md`。

## 素材要求

- 视频导出为 SRT 字幕文件，纯文本格式，AI 可直接读取
- 按主题分类存放（学习方法、赚钱创业、认知思维、自媒体创作等，分类数量根据实际情况定）
- 每批处理 ≤30 条，适配 AI 上下文窗口
